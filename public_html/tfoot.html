Tag HTML <tfoot>

Exemplo

Uma tabela HTML com um elemento <thead>, <tbody> e <tfoot>:

<table>
 <thead>
   <tr>
      <th>Mês</th>
      <th>Poupança</th>
  </tr>
 </thead>
  <tbody>
    <tr>
      <td>Janeiro</td>
      <td>R$100</td>
    </tr>
    <tr>
      <td>Fevereiro</td>
      <td>R$80</td>
  </tr>
  </tbody>
 <tfoot>
    <tr>
      <td>Soma</td>
      <td>R$180</td>
    </tr>
 </tfoot>
</table>

Definição e uso

A tag <tfoot> é usada para agrupar o conteúdo do rodapé em uma tabela HTML.


O elemento <tfoot> é usado em conjunto com os elementos <thead> e <tbody> para especificar cada parte de uma tabela (rodapé, cabeçalho, corpo).


Os navegadores podem usar esses elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e rodapé. Além disso, ao imprimir uma tabela grande que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé da tabela sejam impressos na parte superior e inferior de cada página.


Nota: O elemento <tfoot> deve ter uma ou mais tags <tr> dentro.


A tag <tfoot> deve ser usada no seguinte contexto: Como filho de um elemento <table>, após qualquer elemento <caption>, <colgroup>, <thead> e <tbody>.


Dica: Os elementos <thead>, <tbody> e <tfoot> não afetarão o layout da tabela por padrão. No entanto, você pode usar CSS para estilizar esses elementos.

Atributos

A tag <tfoot> suporta os atributos globais em HTML.


A tag <tfoot> oferece suporte aos atributos de evento em HTML.

Outros exemplos

Exemplos

Estilizando <thead>, <tbody> e <tfoot> com CSS:

<html>
<head>
<style>
thead {color: green;}
tbody {color: blue;}
tfoot {color: red;}

table, th, td {
  border: 1px solid black;
}
</style>
</head>
<body>

<table>
  <thead>
    <tr>
      <th>Mês</th>
      <th>Poupança</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Janeiro</td>
      <td>R$100</td>
    </tr>
    <tr>
      <td>Fevereiro</td>
      <td>R$80</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Soma</td>
      <td>R$180</td>
    </tr>
  </tfoot>
</table>
</body>
</html>

Como alinhar o conteúdo dentro de <tfoot> (com CSS):

<table style="width:100%">
  <tr>
    <th>Mês</th>
    <th>Poupança</th>
  </tr>
  <tr>
    <td>Janeiro</td>
    <td>R$100</td>
  </tr>
  <tr>
    <td>Fevereiro</td>
    <td>R$80</td>
  </tr>
  <tfoot style="text-align:center">
    <tr>
      <td>Soma</td>
      <td>R$180</td>
    </tr>
  </tfoot>
</table>

Como alinhar o conteúdo verticalmente dentro de <tfoot> (com CSS):

<table style="width:100%">
  <tr>
    <th>Mês</th>
    <th>Poupança</th>
  </tr>
  <tr>
    <td>Janeiro</td>
    <td>R$100</td>
  </tr>
  <tr>
    <td>Fevereiro</td>
    <td>R$80</td>
  </tr>
  <tfoot style="vertical-align:bottom">
    <tr style="height:100px">
      <td>Soma</td>
      <td>$180</td>
    </tr>
  </tfoot>
</table>
