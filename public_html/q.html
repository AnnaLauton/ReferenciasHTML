Tag HTML <q>

Exemplo

Marca uma citaçao curta:

<p>Como dizia Einstein:
<q>A vida é igual a andar de bicicleta. Para manter o equilíbrio é preciso se manter em movimento.</q></p>

Definição e uso

A tag <q> define uma citaçao curta.


Navegadores normalmente inserem aspas antes e depois da citação.


Dica: Use <blockquote>para citações longas.

Atributos

Atributo Valor Descrição
cite URL Especifica a URL fonte da citação

A tag <q> tambem suporta os Atributos Globais no HTML.


A tag <q> tambem suporta os Atributos de Eventos no HTML.


Outros exemplos
