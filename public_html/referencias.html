Tag Descrição
<address> Define informações de contato para o autor/proprietário de um documento ou artigo
<aside> Define conteudo à parte sobre o contexto em que esta localizada
<br> Insere uma única quebra de linha
<data> Adiciona uma tradução legível por máquina de um determinado conteúdo
<footer> Define um rodapé para um documento ou uma seção
<head> Contém metadados / informações para o documento
<meter> Define uma medição escalar dentro de uma faixa conhecida ou um valor fracionário.
<nav> Define um conjunto de links de navegação.
<output> Define o resultado de um cálculo
<param> Define parametros para um elemento <object>
<q> Define uma citaçao curta
<summary> Define um título visível para o elemento <details>
<svg> Define um contêiner para gráficos SVG
<textarea> Define um controle de entrada multilinha (área de texto)
<tfoot> Agrupa o conteúdo do rodapé em uma tabela
<tr> Define uma linha numa tabela HTML
<track> Define faixas de texto para elementos de mídia (<video> e <audio>)
<video> É usada para incorporar conteúdo de vídeo em um documento
