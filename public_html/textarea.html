Tag HTML <textarea>

Exemplo

Um controle de entrada de texto de várias linhas (text area):

<label for="exemp">Exemplo:</label>

<textarea id="exemp" name="exemp" rows="4" cols="50">
 Aqui mostramos as referências para HTML5.
</textarea>

Definição e uso

A tag <textarea> define um controle de entrada de texto de várias linhas.


O elemento <textarea> é frequentemente usado em um formulário, para coletar entradas do usuário, como comentários ou análises.


Uma textarea pode conter um número ilimitado de caracteres e o texto é renderizado em uma fonte de largura fixa (geralmente Courier).


O tamanho de uma área de texto é especificado pelos atributos <cols> e <rows> (ou com CSS).


O atributo name é necessário para referenciar os dados do formulário após o formulário ser enviado (se você omitir o atributo name, nenhum dado da textarea será enviado).


O atributo id é necessário para associar a textarea a um rótulo.


Dica: Sempre adicione a tag <label> para melhores práticas de acessibilidade!

Atributos

Atributo Valor Descrição
autofocus autofocus Especifica que uma textarea deve receber foco automaticamente quando a página carrega
cols number Especifica a largura visível de uma textarea
dirname textareaname.dir Especifica que a direção do texto da textarea será enviada
disabled disabled Especifica que uma textarea deve ser desativada
form form_id Especifica a qual formulário a área de texto pertence
maxlength number Especifica o número máximo de caracteres permitidos na área de texto
name text Especifica um nome para uma área de texto
placeholder text Especifica uma pequena dica que descreve o valor esperado de uma área de texto
readonly readonly Especifica que uma área de texto deve ser somente leitura
required required Especifica que uma área de texto é necessária / deve ser preenchida
rows number Especifica o número visível de linhas em uma área de texto
wrap hard
soft		 Especifica como o texto em uma área de texto deve ser quebrado quando submetido em um formulário
