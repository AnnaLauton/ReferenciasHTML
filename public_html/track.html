Tag HTML <track>

Exemplo

O exemplo a seguir exibe um vídeo com faixas de legendas para dois idiomas:

<video width="320" height="240" controls>
  <source src="filme.mp4" type="video/mp4">
  <source src="filme.ogg" type="video/ogg">
  <track src="filme.vtt" kind="subtitles" srclang="en" label="English">
  <track src="filme.vtt" kind="subtitles" srclang="no" label="Norwegian">
</video>

Definição e uso

A tag <track> especifica faixas de texto para elementos <audio> ou <video>.


Este elemento é usado para especificar legendas, arquivos de legendas ou outros arquivos contendo texto, que devem ser visíveis durante a reprodução da mídia.


Tracks são formatadas no formato WebVTT (arquivos .vtt).

Atributos

Atributo Valor Descrição
defaut default Especifica o track que deve ser habilitado se as preferências do usuário não indicarem que outro track seria mais apropriado
kind captions
chapters
descriptions
metadata
subtitles		 Especifica o tipo da track de texto
label text Especifica o título da track de texto
src URL Especifica a URL do arquivo da track
srclang language_code Especifica o idioma dos dados de texto do track (obrigatório se kind = "subtitles")

Outros exemplos

Exemplos

Este exemplo é em relação a um vídeo:

<video controls src="video.mp4">
 <track default kind="captions" srclang="en"
 src="video.vtt" />
 Desculpa, seu navegador não suporta!
</video>

Este exemplo é em relação a um audio:

<audio controls>
 <source src = "audio.mp3">
 <track src = "subtitles.vtt" kind = "subtitles" srclang = "en" label =" English">
 <p>:Desculpa, seu navegador não suporta!</p>
</audio>
