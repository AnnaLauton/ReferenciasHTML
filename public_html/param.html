Tag HTML <param>

Exemplo

Defina o parametro "autoplay" como "true", para que o som toque assim que a página carregar:

<object data="horse.wav">
 <param name="autoplay" value="true">
</object>

Definição e uso

A tag <param> é usada para definir parametros para um elemento <object>.


Atributos

Atributo Valor Descrição
name name Especifica o nome de um parâmetro
value value Especifica o valor de um parâmetro

A tag <param> tambem suporta os Atributos Globais no HTML.


A tag <param> tambem suporta os Atributos de Eventos no HTML.


Outros exemplos

Define o valor do parametro "controller" como "true" para o objeto

<object data="https://www.youtube.com/embed/JHq3pL4cdy4">
 <param name="controller" value="true">
</object>

Define o valor do parametro "video" para "play"

<object data= "https://media.geeksforgeeks.org/wp-content/uploads/20210209234048/InShot_20210209_233841711.mp4">  <param name="video" value="play">
</object>
