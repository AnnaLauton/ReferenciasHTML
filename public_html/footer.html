Tag HTML <footer>

Exemplo

Uma seção de rodapé em um documento:

<footer>
 <p>Autor:Joao Silva</p>
 <p><a href="joao@email.com">joao@email.com</a></p>
</footer>

Definição e uso

A tag <footer> define um rodapé para um documento ou uma seção.


Um elemento <footer> tipicamente contém:

  - Informação sobre o autor

  - Informação sobre os direitos autorais

  - Informações de contato

  - Mapa do Site

  - Links de volta pro topo

  - Documentos Relacionados


Podem haver varios elementos <footer> em um mesmo documento.


Atributos

A tag <footer> suporta os Atributos Globais no HTML.


A tag <footer> tambem suporta os Atributos de Eventos no HTML.


Outros exemplos

Informaçao no rodapé de um site

<footer>
 <nav>
 <ul>
 <li><a href="#">Sobre o autor</a></li>
 <li><a href="#">Mapa do Site</a></li>
 <li><a href="#">Entre em contato</a></li>
 </ul>
 </nav>
</footer>

Informaçao que aparece no espaço do rodapé

<footer>
 <p>Texto que aparece dentro do rodapé</p>
</footer>
