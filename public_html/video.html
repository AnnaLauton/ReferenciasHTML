Tag HTML <video>

Exemplo

Reproduza um vídeo:

<video width="320" height="240" controls>
 <source src="movie.mp4" type="video/mp4">
 <source src="movie.ogg" type="video/ogg">
 Your browser does not support the video tag.
</video>

Definição e uso

A tag <video> é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo.


A tag <video> contém uma ou mais tags <source> com diferentes fontes de vídeo. O navegador escolherá a primeira fonte que suporta.


O texto entre as tags <video> e </video> só será exibido em navegadores que não suportam o elemento <video>.


Existem três formatos de vídeo suportados em HTML: MP4, WebM e OGG.


Atributos

<
Atributo Valor Descrição
autoplay autoplay Especifica que o vídeo começará a ser reproduzido assim que estiver pronto
controls controls Especifica que os controles de vídeo devem ser exibidos (como um botão reproduzir / pausar etc.).
height pixels Define a altura do player de vídeo
loop loop Especifica que o vídeo começará de novo, toda vez que for concluído
muted muted Especifica que a saída de áudio do vídeo deve ser silenciada
poster URL Especifica uma imagem a ser mostrada durante o download do vídeo ou até que o usuário pressione o botão de reprodução
preload auto
metadata
none		 Especifica se e como o autor acha que o vídeo deve ser carregado quando a página carrega
src URL Especifica o URL do arquivo de vídeo
width pixels Define a largura do player de vídeo

Outros exemplos

Exemplo

Um exemplo de vídeo com legendas:

<video src="foo.ogg">
 <track kind="subtitles" src="foo.en.vtt" srclang="en" label="English">
 <track kind="subtitles" src="foo.sv.vtt" srclang="sv" label="Svenska">
</video>

Exemplo simples de vídeo:

<video src="arquivovideo.ogg" autoplay poster="imagemprevia.jpg">
 Desculpa, o seu navegador não suporta vídeos incorporados,
 mas você pode <a href="videofile.ogg">baixá-lo</a>
 e assistir pelo seu reprodutor de mídia favorito!
</video>
