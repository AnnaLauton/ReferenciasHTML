Tag HTML <meter>

Exemplo

Use o elemento medidor para medir dados dentro de um determinado intervalo (um medidor):

<label for="disk_c">Disk usage C:</label>
<meter id="disk_c" value="2" min="0" max="10">2 out of 10</meter>

<label for="disk_d">Disk usage D:</label>
<meter id="disk_d" value="0.6">60%</meter>

Definição e uso

A tag <meter> define uma medição escalar dentro de uma faixa conhecida ou um valor fracionário. Isso também é conhecido como medidor.

Exemplos: uso do disco, relevância do resultado de uma consulta, etc.


Observação: A tag <meter> não deve ser usada para indicar o progresso (como em uma barra de progresso). Para barras de progresso, use a tag <progress>.


Dica:Sempre adicione a tag <label> para melhores práticas de acessibilidade!

Atributos

<
Atributo Valor Descrição
form form_id Especifica a qual formulário o elemento <meter> pertence
high number Especifica o intervalo que é considerado um valor alto
low number Especifica o intervalo que é considerado um valor baixo
max number Especifica o valor máximo do intervalo
min number Especifica o valor mínimo do intervalo. O valor padrão é 0
optimum number Especifica qual valor é o valor ideal para o medidor
value number Requerido. Especifica o valor atual do medidor
