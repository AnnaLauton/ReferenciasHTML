Tag HTML <summary>

Exemplo

Usando o elemento <summary>:

<details>
 <summary>Epcot Center</smmary>
  <p>Epcot é um parque temático no Walt Disney World Resort com atrações emocionantes, pavilhões
  internacionais, fogos de artifício premiados e eventos especiais sazonais.</p>
</details>

Definição e uso

A tag <summary> define um título visível para o elemento <details>.
O título pode ser clicado para visualizar / ocultar os detalhes.

Observação: o <summary> elemento deve ser o primeiro elemento filho do elemento <details>.

Outros exemplos

Exemplo

Um exemplo que usa a tag <summary>

<details>
 <summary>Alguns detalhes</summary>
 <p>Mais informações sobre os detalhes.</p>
</details>

Use CSS para definir o estilo de <details> e <summary>:

<html>
 
<style>
  details > summary {
   padding: 4px;
   width: 200px;
   background-color: #eeeeee;
   border: none;
   box-shadow: 1px 1px 2px #bbbbbb;
   cursor: pointer;
 }
  details > p {
   background-color: #eeeeee;
   padding: 4px;
   margin: 0;
   box-shadow: 1px 1px 2px #bbbbbb;
 }
</style>

<body>
  <details>
   <summary>Epcot Center</summary>
   <p>Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.</p>
  </details>
</body>

</html>
