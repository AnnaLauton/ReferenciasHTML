Tag HTML <nav>

Exemplo

Usando o elemento <nav>:

<nav>
 <a href="Referencia.html"><summary></a>
 <a href="Sobre.html">Sobre</a>
</nav>.

Definição e uso

A tag <nav> define um conjunto de links de navegação.

Observe que não são todos os links de um documento devem estar dentro de um elemento <nav>.

O elemento <nav> destina-se apenas ao bloco principal de links de navegação. Navegadores, como leitores de tela para usuários com deficiência, podem usar este elemento para determinar se deve omitir a renderização inicial deste conteúdo.
