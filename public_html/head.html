Tag HTML <head>

Exemplo

Um documento HTML simples, com uma tag <title> dentro da seção head:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <title>Exemplo</title>
</head>
<body>

<p>Este é um paragrafo.</p>

</body>
</html>

Definição e uso

O elemento <head> é um contêiner para metadados (dados sobre dados) e é colocado entre a tag <html> e a tag <body>


Metadados são dados sobre o documento HTML. Os metadados não são exibidos.


Os metadados normalmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras meta informações.


Os seguintes elementos podem ir dentro do elemento <head>:

  - <title> (exigido em cada documento HTML)
  - <style>
  - <base>
  - <link>
  - <meta>
  - <script>
  - <noscript>

Atributos

A tag <head> suporta os atributos globais em HTML.

Outros exemplos

Exemplos

A tag <base> (especifica uma URL padrão e destino para todos os links em uma página) vai para <head>:

<html>
<head>
  <base href="https://www.w3schools.com/" target="_blank">
</head>
<body>

<img src="images/stickman.gif" width="24" height="39" alt="Stickman">
<a href="tags/tag_base.asp">Tag base HTML</a>

</body>
</html>

A tag <style> (adiciona informações de estilo a uma página) vai dentro de <head>:

<html>
<head>
  <style>
    p {color:blue;}
  </style>
</head>
<body>

<p>Um parágrafo.</p>

</body>
</html>

A tag <link> (links para uma folha de estilo externa) vai dentro de <head>:

<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>

<p>Um exemplo!</p>

</body>
</html>
