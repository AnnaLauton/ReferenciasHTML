Tag HTML <output>

Exemplo

Realizando um cálculo e mostrando o resultado em um elemento <output>

<form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
  <input type="range" id="a" value="50">
  +<input type="number" id="b" value="25">
  =<output name="x" for="a b"></output>
</form>

Definição e uso

A tag <output> é utilizada para representar o resultado de um cálculo (como um executado por um script).

Atributos

Atributo Valor Descrição
for element_id Especifica a relação entre o resultado do cálculo e os elementos usados ​​no cálculo
form form_id Especifica a qual formulário o elemento de saída pertence
name name Especifica um nome para o elemento de saída

Exemplos

Neste exemplo, a tag <output> é usada com o atributo for e form.

<form oninput="sumresult.value = parseInt(A.value) + parseInt(B.value) + parseInt(C.value)">
 <input type="number" name="A" value="50" /> +
 <input type="range" name="B" value="0" /> +
 <input type="number" name="C" value="50" />
 <br /> Submit Result:
 <output name="sumresult" for="A B C"> </output>
 <br>
 <input type="submit">
</form>
