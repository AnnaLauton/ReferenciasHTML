Tag HTML <tr>

Exemplo

Uma tabela HTML simples com 3 linhas , uma linha de cabeçalho e duas linhas de dados

<table>
 <tr>
  <th>Mes</th>
  <th>Economias</th>
 </tr>
 <tr>
  <td>Janeiro</td>
  <td>$100</td>
 </tr>
 <tr>
  <td>Fevereiro</td>
  <td>$80</td>
 </tr>
</table>

Definição e uso

A tag <tr> define uma linha numa tabela HTML.


Um elemento <tr> contem um ou mais elementos <th> ou <td>.


Atributos


A tag <tr> tambem suporta os Atributos Globais no HTML.


A tag <tr> tambem suporta os Atributos de Eventos no HTML.


Outros exemplos

Criaçao de cabeçalhos de tabela:

<table>
  <tr>
    <th>Nome</th>
    <th>Email</th>
    <th>Telefone</th>
  </tr>
  <tr>
    <td>Joao</td>
    <td>joao@gmail.com</td>
    <td>123-45-678</td>
  </tr>
</table>

Criaçao de tabela com legenda

<table>
  <caption>Economias Mensais</caption>
  <tr>
    <th>Mes</th>
    <th>Economias</th>
  </tr>
  <tr>
    <td>Janeiro</td>
    <td>$100</td>
  </tr>
  <tr>
    <td>Fevereiro</td>
    <td>$80</td>
  </tr>
</table>
