Tag HTML <address>

Exemplo

Informaçoes de contato para Exemplo.com

<address>
 Escrito por <a href="email@exemplo.com> Maria </a>.<br>
 Visite nos em:<br>
 Exemplo.com<br>
 Primeiro andar, Palacio do Planalto<br>
 Brasil
</address>

Definição e uso

A tag <address> define informações de contato para o autor/proprietario de um documento ou artigo.


A informação de contato pode ser um endereço de email, URL, endereço físico, numero de telefone, midias sociais, etc.


O texto no elemento <address> normalmente é renderizado em italico, e navegadores sempre adicionarão uma quebra de linha antes e depois do elemento <address>.


Atributos

A tag <address> suporta os Atributos Globais no HTML.


A tag <address> tambem suporta os Atributos de Eventos no HTML.


Outros exemplos

O exemplo abaixo mostra um meio de contato para o site Blue Bezer

<address>
 <p><a href="https://bluebezer.com.br/contato">Blue Bezer</a></p>
</address>

O exemplo abaixo mostra um meio de contato para o MIT

<address>
 <p><a href="http://www.w3.org/Consortium/contact-mit">MIT</a></p>
</address>
